# StudyCards

Lernkarten-Anwendung mit Java und JavaFX.

## Voraussetzungen

- Java 17, Maven, MySQL

## Starten

```
mvn javafx:run
```

## Status

- [x] Datenmodelle und Datenbankverbindung
- [x] DAO-Klassen
- [x] Hauptansicht
- [x] Detailansicht (Kartenverwaltung)
- [x] Lernmodus (teilweise)
- [ ] Ergebnis-Anzeige am Ende
